<h1 align="center">All for one</h1>

## :memo: Descrição
Projeto com o codinome 'All for One', em que é praticado alguns conceitos de SQL.

## :books: Funcionalidades
* SELECT: Exibir dados.
* INSERT INTO: Adicionar dados.
* UPDATE: Atualizar dados.
* DELETE: Deletar dados.
